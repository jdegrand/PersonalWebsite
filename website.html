RFID Music Player

RFID has something that has always seemed interesting to me, and I had never real knew much about it before this project. I wanted to see if I can assign different songs to play when swiping different RFID cards. Using a Raspberry Pi 3 with an RFID Module and a Python library, I was able to turn this into a reality. I started this project my first semester at RIT, when I didn't have much programming experience. Now that I have a bit more experience, I hope to add more to it in the future.
